### Week 2: Cutting and 2D Design <br> <p style="color:rgb(48, 48, 255)"><font size="+2">This week, we began using the laser cutting machine in the lab and Fusion360 in order to make press-fit shapes.</font></p> <br> <p style="color:rgb(48, 48, 255)"><font size="+2">When beginning my press-fit shape, I used the parameters tool provided by Fusion360 in order to set my building parameters, as shown below.</font></p> <style> .center { display: block; margin-left: auto; margin-right: auto; width: 50%; } </style> <img src="./pressfitparameters.png" width="400" height="242" alt="parameters" class="center"> <br> <p style="color:rgb(48, 48, 255)"><font size="+2">After setting my parameters, I created a 50mm by 50mm square to start. One of my first steps was to create construction lines using the side menu and creating two diagonal lines across the middle of my square, in order to improve my accuracy when creating my shape. For my shape, I wanted to create a square that had notches in the corners, thus allowing for some rather abstract shapes whilst keeping four notch points. Afterwards, I offset the horizontal mid-line by using the measurement of "thickness/2". This created the gaps that would soon become my notches. The image of my first sketch is shown below.</font></p> <style> .center { display: block; margin-left: auto; margin-right: auto; width: 50%; } </style> <img src="./pressfitfirstsketch.png" width="400" height="242" alt="firstsketch" class="center"> <br> <p style="color:rgb(48, 48, 255)"><font size="+2">After creating my first sketch, I then adjusted the diagonal notch width for kerf, thus closing up the notches enough that the kerf would not make my notches loose. </font></p> <style> .center { display: block; margin-left: auto; margin-right: auto; width: 50%; } </style> <img src="./pressfitsketchwithkerf.png" width="400" height="242" alt="kerfsketch" class="center"> <br> <p style="color:rgb(48, 48, 255)"><font size="+2">Then, I used the extrude tool by pressing E and selected all of the sections of my shape except for the notches and the small corner pieces that were adjacent to them, in order to create the square notch seen here:</font></p> <style> .center { display: block; margin-left: auto; margin-right: auto; width: 50%; } </style> <img src="./first_extrusion.png" width="400" height="242" alt="firstextrusion" class="center"> <br> <p style="color:rgb(48, 48, 255)"><font size="+2">After extruding, the next and final two steps were to fillet the corners of my notches.
